# Troubleshooting

## Common Issues

| Symptom | Cause | Solution |
|---------|-------|----------|
| Module not found | Missing build step | Run `npm run build` |
| API key error | Missing env var | Set required API key in `.env` |
| Tests fail | Outdated deps | Run `npm install` |
