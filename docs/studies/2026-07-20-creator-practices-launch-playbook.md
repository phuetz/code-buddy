# Chasse aux pratiques créateurs + playbook de lancement (2026-07-20, soir)

Recherche ciblée YouTube/tutoriels/retours de créateurs AI-lookbook. Les 7
points qui CHANGENT nos décisions :

1. **La variation éditoriale est une exigence de survie, pas un plus** :
   YouTube évalue désormais LA CHAÎNE entière (16 chaînes AI-slop supprimées
   01/2026) ; « slight variations between videos » = démonétisation. Deux
   Shorts consécutifs ne doivent jamais partager le même trio {lieu, format
   sériel, structure narrative} + une couche éditoriale par vidéo (texte
   narratif, micro-histoire). Notre gate anti-répétition devient un outil de
   conformité YPP autant que de qualité.
2. **Deux masters par vidéo** : ~13 s (sweet spot Reels/TikTok) ET ~20-25 s
   bouclé pour Shorts (frame finale ≈ frame 1, musique continue au point de
   boucle, zéro carte de fin ; test = 3 visionnages en boucle). Le hook se joue
   à 1 seconde (pas 3) : mouvement dès la frame 1 + texte 3-7 mots tiers
   supérieur lisible en muet.
3. **Musique** : Epidemic OK sur Shorts/Reels (claims sans effet visibilité/
   revenus) ; sur TikTok utiliser les sons montants NATIFS de la plateforme.
4. **Titres recherchables** (changement 01/2026 : les Shorts sont indexés) :
   requête plausible + numérotation sérielle ; 1-3 hashtags + #shorts ;
   heure fixe (20-21 h audience) et ne plus y penser.
5. **Interdits absolus** : supprimer/re-uploader un Short ; >1/jour ;
   watermarks ; engagement-bait ; réponse hors-personnage.
6. **Funnel** : PPV Fanvue seulement après ~100 abonnés (le lancer tôt tue
   l'urgence) ; split 85/15 ; SFW lifestyle viable ; la communauté fan-à-fan
   pèse plus que le personnage (terminer par une question à choix « tenue 2 ou
   4 ? », pinned comment = pont Instagram). Character bible écrite (backstory,
   lieux, voix textuelle) avant J1. Disclosure fière (#AI) : le cas Pellegrini
   montre que ça ACCÉLÈRE la croissance.
7. **Patience calibrée** : seed de test 50-150 viewers/24-72 h ; premier 100K+
   typiquement entre l'upload 30 et 50 ; les Shorts >28 jours meurent (fenêtre
   glissante) → décision go/no-go de format à J45-50 ; à J8 analyser
   « Viewed vs Swiped » (<30 % swipe-through = refaire le hook).

## Checklist « 30 premiers jours » (résumé opérationnel)

- Avant J1 : character bible, banque de 10-14 Shorts gatés anti-répétition,
  2 masters/vidéo, spec boucle + hook, disclosure partout.
- S1 : 1/jour heure fixe, titres recherchables sériels, question à choix en
  fin, Reels 13 s + TikTok son natif.
- S2-3 : répliquer la STRUCTURE des 2 meilleurs (jamais le contenu), replay
  ≥10 %, réponses en Lisa <2 h, pas de PPV.
- S4 : suite numérotée <48 h sur tout Short qui décolle ; bilan matrice
  format × lieu × durée ; variantes fraîches des tops >28 j.
